# User Story — Création d'une entreprise

En tant que représentant d'une entreprise,

Je souhaite créer mon compte AllenVoice,

## Critères d'acceptation

* L'entreprise doit avoir un nom.
* L'entreprise doit avoir une adresse email unique.
* L'entreprise doit fournir un numéro de téléphone.
* Une configuration IA par défaut doit être créée automatiquement.
* La date de création doit être enregistrée.

Afin de configurer mon agent vocal IA et de gérer mes appels depuis la plateforme.
